Portfolio Website - User Interfaces

De opdracht was om voor User Interfaces een portfolio-website te maken. Zo konden we oefenen op HTML en CSS en een beetje design. Deze opdracht staat op 50% van het eindtotaal. Deze opdracht was een gevecht. Eerst moesten we zelf voor een design zorgen, iets wat voor mij moeilijk was. Ik kon wat in mijn hoofd zat niet op papier krijgen, laat staan een scherm. Uiteindelijk heb ik de hulp ingeroepen van mijn lieve zus, Laura, die grafisch design heeft gestudeerd. Zij heeft mij op weg geholpen naar een simpele wireframe en wat tips over hoe deze opdracht aan te pakken. Toch was ik de eerste keer niet tevreden; de code was niet meer leesbaar en ik was niet blij over het uitzicht. Daarnaast heeft het lang geduurd voor er iets op scherm stond, op de juiste plaats. Deze keer ging het iets vlotter, ik ben begonnen met de simpele onderdelen en zo verdergewerkt naar het moeilijkere. Nu kan ik, hopelijk, iets vlotter overweg met HTML en CSS.

Eventuele aanpassingen voor later: een slideshow bij projecten i.p.v. allemaal aparte foto's.
